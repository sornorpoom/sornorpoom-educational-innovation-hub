## ⚡ คุณสมบัติหลักที่อัปเดตแบบ Real-time (Dynamic Integration)

> [!IMPORTANT]
> **ระบบนี้ใช้โครงสร้างสถาปัตยกรรมแบบ Real-time Data Binding:**
> เมื่อมีการแก้ไข เพิ่มลบ หรือเปลี่ยนแปลงข้อมูลใด ๆ ใน **Google Sheets** หน้าเว็บแอปพลิเคชันจะอัปเดตข้อมูลและแสดงผลหน้าการ์ดนวัตกรรมล่าสุดทันทีในการเปิดใช้งานครั้งถัดไป **โดยผู้พัฒนาไม่จำเป็นต้องแก้ไขหรืออัปโหลดไฟล์โค้ดใด ๆ บน GitHub เพิ่มเติม**

*   **Real-time Google Sheet Sync:** เชื่อมต่อและดึงข้อมูลสดจากแผ่นตารางผ่าน API ของ Google Workspace
*   **Offline Fallback Mode:** มีระบบฐานข้อมูลสำรองบรรจุไว้ภายในตัวโค้ด หากระบบเครือข่ายของ Google Sheet มีความขัดข้องหรือเกิดข้อผิดพลาดด้านสิทธิ์เข้าใช้งาน หน้าเว็บจะยังสามารถเปิดทำงานแสดงข้อมูลล่าสุดจากไฟล์สำรองได้เสมอ
*   **Accessibility Design (สำหรับครูผู้สอน):**
    *   **font size Adjustment:** เลือกปรับขนาดข้อความได้ 3 ระดับ (เล็ก, กลาง, ใหญ่) เพื่อความสบายตาในการอ่านข้อมูล
    *   **Theme Controller:** เลือกสลับสีได้ 3 ธีมสี (สว่าง / อบอุ่นถนอมสายตา / มืดสำหรับเวลากลางคืน)
*   **QR Code Generator:** ระบบจะสร้าง QR Code อัตโนมัติในหน้ารายละเอียดนวัตกรรมแต่ละรายการ เพื่อให้คุณครูสแกนผ่านโทรศัพท์มือถือเข้าสู่ระบบได้ทันที
*   **Responsive Grid:** ปรับเปลี่ยนรูปแบบเลย์เอาต์อัตโนมัติเพื่อให้แสดงผลสวยงามบนมือถือ แท็บเล็ต และเดสก์ท็อป

## 📁 โครงสร้างโปรเจกต์ (Project Files)

ในการอัปโหลดขึ้น GitHub บัญชีของคุณจะมีไฟล์เหล่านี้เป็นหลัก:

*   `code.gs` - โค้ดส่วนหลังบ้าน (Apps Script) ทำหน้าที่ควบคุม Route และการดึงข้อมูลจาก Google Sheets
*   `index.html` - โค้ดส่วนแสดงผลหน้าจอ (Frontend) การจัดการธีม ขนาดฟอนต์ ค้นหา ตัวกรอง และโมดัลหน้าต่างข้อมูล
*   `README.md` - คำอธิบายรายละเอียดและวิธีใช้งานระบบ (ไฟล์นี้)
*   `.gitignore` - ไฟล์กำหนดไม่ให้อัปโหลดไฟล์ที่ไม่จำเป็นหรือไฟล์ส่วนตัวขึ้น GitHub
